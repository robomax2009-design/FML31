Затраты. Виды затрат.

1. Зависимость от объема производства.

  • FC (Fixed Costs) — Постоянные затраты — не зависят от объёма выпуска.
    FC = const
  • VC (Variable Costs) — Переменные затраты — меняются пропорционально объёму производства.
    VC = f(Q)
  • TC (Total Costs) — Общие затраты — сумма постоянных и переменных издержек.
    TC = FC + VC

2. Средние затраты на единицу продукции

  • AFC (Average Fixed Costs) — Средние постоянные затраты — постоянные затраты на единицу продукции. Снижаются с ростом Q.
    AFC = FC / Q
  • AVC (Average Variable Costs) — Средние переменные затраты — переменные затраты на единицу продукции.
    AVC = VC / Q
  • ATC (AC) (Average Total Costs) — Средние общие затраты — полная себестоимость единицы продукции.
    AC = TC / Q = AFC + AVC

3. Предельные и специфические затраты

  • MC (Marginal Costs) — Предельные затраты — прирост затрат при производстве одной дополнительной единицы.
    MC = ΔTC / ΔQ
