Определения

  • Химия — Одна из основных наук естествознания, изучающая внутренний состав, внутреннее строение материи, закономерности качественных изменений, разложения и превращения веществ, а также закономерности образования новых веществ в результате качественных изменений

    • Химия отличается от физики изменением состава вещества.

  • Простое вещество — состоит из одного типа элементов.
  • Сложное вещество — состоит из нескольких типов элементов.
  • Гидроксогруппа — (OH)y
  • Химическая реакция — превращение реагентов в новые вещества.
  • Энергия активации — минимальная энергия для реакции.
  • Эндотермические реакции — сопровождаются поглощением тепла.
  • Экзотермические реакции — сопровождаются выделением тепла.
  • Тепловой эффект — количество теплоты, выделяемое или поглощаемое.

Степени окисления

  1. Простые вещества: 0
  2. Фтор: всегда −1
  3. Кислород: обычно −2
  4. Водород: +1 с неметаллами, −1 с металлами
  5. Сумма степеней окисления в молекуле всегда 0

Валентность

Валентность оксида ЭxOy вычисляется как: 2· y / x

Химические реакции

Кислоты

  • Кислота + основание → соль + H₂O
  • Кислота + осн. оксид → соль + H₂O
  • Кислота + металл → соль + H₂

Основания

  • Основание + кислота → соль + H₂O
  • Щёлочь + соль → новая соль + новое основание

Соли

  • Соль + кислота → новая кислота + новая соль
  • Соль + металл → новая соль + металл
