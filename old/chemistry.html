    Определения
  • Химия -
  • Простое в-во - в-во, состоящее из одного типа эл-тов.
  • Сложное в-во - в-во, состоящее из несккольких типов эл-тов.
  • Гидроксогруппа - (OH)y
  • Химическая реакция - превращение одного или нескольких исходных веществ (реагентов) в отличающиеся по химическому составу или строению новый вещества (продукты реакции).
  • Энергия активации - это минимальная энергия, которая необходима молекулам, чтобы вступить в химическо взаимодействие.
  • Эндотермические реакции - хим. реакции, сопровождающиеся поглощением тепла.
  • Экзотермические реакции - химю реакции, сопровождающиеся выделением тепла.
  • Тепловой эффект - количество теплоты, выделяемое или поглощаемое при реакции.

    Химические реакции:

    • Кислот

    1. + основание = соль + вода (Обмен)
    2. + осн. оксид = соль + вода (Обмен)
    3. + амф. оксид = соль + вода (Обмен)
    4. + соль = новая кислота + новая соль (Обмен)
    5. + металл = соль + H2 (Замещение)

    • Оснований

    1. Осн. + кослота = соль + вода (Обмен)
    2. Щёлочь + кисл. оксид = соль + вода (Обмен)
    3. Щёлочь + амф. оксид = соль + вода (Обмен)
    4. Нерастворимое осн. = оксид металла + вода (Разложение)
    5. Щёлочь + металл = соль + H2 (Замещение)
    6. Соль + щёлочь = новая соль + новое основание (Обмен)

    • Солей

    1. + кислота = новая кислота + новая соль (Обмен)
    2. + щёлочь - новая соль + новое основание (Обмен)
    3. Соль1 + соль2 = соль3 + соль4 (Обмен)
    4. Соль + металл = новая соль + новый метал (Замещение)

    • Кислотных оксидов

    1. + вода = кислота (Соединение)
    2. + осн. оксид = ссоль(Соединение)
    3. + фмф. оксид = соль (Соединение)
    4. + щёлочь = соль + вода (Обмен)

    • Основных оксидов

    1. + вода = щёлочь (Соединение)
    2. + кисл. оксид = соль (Соединение)
    3. + амф. оксид = соль (Соединение)
    4. + кислота = соль + вода (Обмен)

    • Амфотерных оксидов

    1. + осн. оксид = соль (Соединение)
    2. + кисл. оксид = соль (Соединение)
    3. + кислота = соль + вода (Обмен)
    4. + щёлочь = соль + вода (Обмен)

    Правила определения Степеней окисления.

  1. С.о. эл-та в простом в-ве равна 0.
  2. С.о. фтора во всех соединениях равна -1 (кроме F2).
  3. С.о. кислорода во всех соединениях равна -2.
    4. Исключения: O2(0), O3(0), OF2(+2, -1), Na2O2(+1, -1), H2O2(+1, -1).
  4. С.о. водорода равнаа +1, если в соединении есть хотя бы один неметалл, иначе -1.
  5. С.о. металла неотрицательна.
    С.о. главных подгрупп всегда положительна, всегда равна номеру группы.
    С.о. побочных подгрупп принимает различные знаачения
  6. Максимальная положительная с.о. равна номеру группы (Исключения: Cu +2, Au +3).
    Минимальная с.о. равна номеру группы - 8.
  7. Сумма с.о. Атомов в молекуле равна 0.
