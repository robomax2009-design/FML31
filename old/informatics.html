Файлы
  • f = open(path, mode="rt", encoding) - открытие файла по относительному/абсолютному пути path. Возвращает файловвый объъект.
      • Режимы (mode):
      Указаны первые буквы для каждого режима. Вторая буква по умолчанию t(text - чтение текстового файла), также есть b(bytes - чтение бинарного файла)
    1. r - Чтение файла. Курсор находится перед первым символом в файле.
    2. w - Создание нового файла для записи. ВАЖНО: если перед открытием файла уже был файл по данному пути, старый будет стёрт.
    3. x - То же, что и w, но вызывает ошибку если файл по данному пути уже существует. Нужно для заадач "дебагга" - как предосторожность до того как закончите программу, случайно не удалить имеющиеся файлы.
    4. a - Открытие файла для записи. Если файл по данному пути уже существует, запись будет происходить в конец этого файла, если не существует - в начало нового файла. P.s. Подойдёт в случает, если вам необходимо чтобы файл сущеествовал несмотря на его содержимое.
      5. Остальные режимы (r+, w+, a+) использовать не принято, они сильно усложняют читаемость кода и не привносят взамен весомой пользы. Для управления курсором можно использовать метод файлового объекта .seek(n) - он перемещает курсор перед указанным символом (нумерация начинается с 0.)
  • .close() - метод файлового объекта. Необходимо вызвать как только закончится алгоритм обработки содержимого

      • Методы файлового объекта для чтения.
    1. .read(n) - Читает n символов и сдвигает курсор на n символов. Если не указать n, вернёт весь файл.
    2. .readline() - Читает файл до переноса строки / конца файла. Симввол переноса строки также возвращается.
    3. .readlines(n) - Выполняет .readline() n раз, возвращает список непустых результатов. Если не указать n, выведет весь файл разделённый на строки (Символ переноса строки входит в элементы списка)
      Методы файлового объекта для записи.
    1. .write(str) - записывает набор данных символов без каких-либо дополнений (без переноса строки)
    2. .writelines(list) - записывает сумму строк из переданного списка в файл.

def merge(a, b): res = [] i = 0 j = 0 while len(a) > i and len(b) > j: if b[j] >= a[i]: res.append(a[i]) i += 1 else: res.append(b[j]) j += 1 res.extend(a[i:]) res.extend(b[j:]) return res def msort(a): if 1 >= len(a): return a k = len(a) // 2 return merge(msort(a[:k]), msort(a[k:]))

def bubble_sort(a): n = len(a) unordered = True while unordered: unordered = False for j in range(n - 1): if a[j] > a[j + 1]: a[j], a[j + 1] = a[j + 1], a[j] unordered = True n -= 1
